# Mysht3x
Sensirion SHT3x温湿度センサ用ライブラリ
Arduino用

計測から読み出しまで時間がかかるので、計測コマンドの発行と読み出しコマンドの発行を分けているため

計測と読み出しの間に長期間プログラムが停止せずにすみます。

自動読み出し関数は1秒間間隔でユーザーが呼び出す必要があります。
